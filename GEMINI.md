# EduConnect - OpenSpec Enabled

Este proyecto utiliza **OpenSpec** para el Desarrollo Basado en Especificaciones (SDD).

## Comandos Disponibles

- `/opsx:explore` - Fase de investigación y lluvia de ideas.
- `/opsx:propose` - Crear una nueva propuesta de cambio (genera `proposal.md`, `design.md`, `tasks.md`).
- `/opsx:apply` - Implementar las tareas de una propuesta.
- `/opsx:archive` - Finalizar y archivar un cambio completado.

## Estructura
- `openspec/specs/` - Especificaciones globales del sistema.
- `openspec/changes/` - Propuestas de cambio activas.
