## ADDED Requirements

### Requirement: Activity History View
El sistema SHALL proporcionar una vista dedicada de "Historial de Actividad" para consultar todas las notificaciones pasadas con filtros por categoría.

#### Scenario: User reviewing past notifications
- **WHEN** un usuario navega a la vista de historial de actividad
- **THEN** el sistema debe mostrar todas las notificaciones (agrupadas o individuales) y permitir filtrarlas por tipo o estado.
