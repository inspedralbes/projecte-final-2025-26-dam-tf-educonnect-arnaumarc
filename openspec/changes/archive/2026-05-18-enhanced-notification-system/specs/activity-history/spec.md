## ADDED Requirements

### Requirement: Full history view
El sistema SHALL proporcionar una vista de página completa donde el usuario pueda navegar por todo el historial de notificaciones.

#### Scenario: Filtering history
- **WHEN** el usuario accede al historial completo
- **THEN** puede filtrar por tipo (Académico, Mensajes, Meet) y estado (Leído/No leído).
